# Super App

A React + Vite Entertainment Dashboard application.

## Features

- User Registration
- Category Selection
- Weather Widget (OpenWeather API)
- News Widget (News API)
- Movie Recommendations (OMDB API)
- Notes Widget
- Timer Widget
- Local Storage Support

## Technologies Used

- React
- Vite
- React Router DOM
- Axios
- CSS

## Installation

```bash
npm install
npm run dev
```

## APIs Used

- OpenWeather API
- News API
- OMDB API
